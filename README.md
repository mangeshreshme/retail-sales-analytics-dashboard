# Retail Sales Analytics Dashboard

This is an end-to-end Retail Sales Analytics Dashboard that I built using **Power BI and MySQL** to analyze business performance across sales, customers, products, and regions.

## Project Overview
In this project, I connected a MySQL retail database to Power BI, cleaned and transformed the data using Power Query, and created a proper data model for analysis.
Also I built interactive dashboards to track key business metrics like revenue, profit, orders, customer behavior, product performance, and growth trends.

## Tools Used
- Power BI
- MySQL
- Power Query
- DAX

## Key Metrics Created
- Total Revenue
- Total Profit
- Total Orders
- Average Order Value
- Profit Margin
- Month-over-Month Growth
- Year-to-Date Revenue
- Repeat Customer Rate
- Total Quantity Sold

## Business Insights Generated
Through this dashboard, I analyzed:
- Identified top-performing regions by revenue and profitability
- Analyzed repeat customer behavior and retention trends
- Compared product performance by revenue, quantity, and profit margin
- Evaluated discount impact on business profitability
- Tracked month-over-month growth and YTD performance
